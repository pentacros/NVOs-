## 8. Other Stylistic Devices

- Consistent **top-of-frame dark gradient overlay** across every shot to protect white/yellow text legibility (darkest near top, fades to transparent by roughly the upper third).
- All text and pill/logo elements are **left-aligned within a consistent safe margin** (~2–3% inset from top/left), appropriate for 9:16 platform UI overlays (avoids being covered by profile/caption UI).
- Visible **stock-footage grain/compression softness** and slight sub-pixel frame jitter (mild handheld/gimbal sway) in most shots.
- No letterboxing/pillarboxing — full-bleed 9:16 throughout.
- Font inconsistency between slides (sans-serif for Block A, serif for Block B, sans-serif again for Block C/CTA) suggests the copy slides were assembled from more than one text template rather than one unified type system — worth normalizing to a single type family when rebuilding programmatically, unless the inconsistency itself is considered "on-brand" for this ad set.
- Color highlight language is consistent even if fonts aren't: **neon/yellow-green** for keyword emphasis in body copy, **royal blue** for all pill/button/badge backgrounds, **white** for primary label text — a usable 3-color accent system for CSS recreation.
