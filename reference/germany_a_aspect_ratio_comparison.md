# Aspect Ratio Comparison — "Germany NVO version A" (germany_1.mp4)

Baseline: 9:16 (1080x1920, 299 frames, 14.97s). Compared against 16:9 (1920x1080) and 1:1 (1080x1080), same duration/frame count — all reviewed frame-by-frame at 20fps.

## 1. Background footage — PILLARBOX WITH BLURRED FILL (confirmed)
Every frame in both 16:9 and 1:1 shows the native vertical footage centered, sharp, occupying a center column (~44% of width in 16:9, wider fraction in 1:1), flanked by a heavily blurred/zoomed duplicate of the same shot filling the side margins. No re-shoot, no alternate framing — same 5 shots (dusk street, dusk intersection, rainy Gherkin street, Berlin Dom, Reichstag), same cut timecodes, in all 3 ratios.

## 2. Text/copy blocks
Content, timing, and highlight styling (script "Germany" gradient, blue pill subtitle, USP bullet cards, gold "Apply for" CTA lead-in) identical across ratios. All text/logos confined to the sharp center column — never extends into the blurred pillarbox bands. Effective font size relative to the *full* wide/square canvas shrinks in 16:9/1:1 since it's the same fixed-pixel layer just reframed.

## 3. Logos (BMW/Bosch/Siemens)
Same reveal order, timing, and position relative to the sharp column across all 3 ratios; no resizing or rearrangement.

## 4. CTA
"Check Eligibility" — same copy, styling, and left-aligned position relative to the sharp column in all 3 ratios. In 1:1, the CTA and preceding "Apply for / Winter 2026 intake" block sit notably in the upper-third of frame (not bottom-anchored) because the square canvas is shorter and the layer's absolute pixel position was preserved rather than re-centered.

## 5. Safe-area / margin rule
Consistent only relative to the embedded sharp footage column (~5-9% left margin, ~5-6% top margin within that column), not as a percentage of the actual delivered canvas. All copy/logos/CTA stay within the ~28-72% width band (16:9) / equivalent inset band (1:1); nothing ever touches the blurred side bands.

## 6. Missing/added/repositioned elements
None — every element present in 9:16 appears identically in 1:1/16:9, just embedded in the pillarboxed strip. No standalone brand/company logo appears in any of the 3 ratios — only the 3 employer logos (BMW/Bosch/Siemens), confined to one proof beat.

**Rebuild implication**: render the 9:16 composite once; produce 1:1/16:9 by centering it as a fixed-pixel column and padding with a blurred/zoomed duplicate of the same footage — no independent re-layout of text/logo/CTA needed for the wider canvases.
